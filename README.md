# Spring Initializr CLI

Scaffold your Spring Boot applications quickly right from your terminal.

---

## Prerequisites

- Node.js 20 or higher installed

---

## Installation

To install globally via npm, run:

```bash
npm install -g @mathstylish/spring-initializr-cli
```

## Usage
```bash
spring
```

## Features

- Choose between Gradle (Groovy/Kotlin) or Maven

- Select Java, Kotlin or Groovy language

- Pick your Spring Boot version

- Configure groupId, artifactId, package name, project name, description, and more

- Select dependencies with interactive autocomplete

- Save and reuse previous configurations to speed up your workflow
